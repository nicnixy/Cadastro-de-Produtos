# Gerenciador de Produtos

Aqui é miniprojeto de gerenciador de produtos. Você escreve as informações pedidas no site e ele armazena as informações e cria um ID automaticamente para eles. Não é linkado a um banco de dados, então não será possível resgatar as informações após a atualização da página.
Este miniprojeto foi feito apenas com HTML, CSS e JavaScript como forma de exercício de fixação sobre os assuntos que estou estudando atualmente
